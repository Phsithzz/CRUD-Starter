# React Product CRUD

A simple **CRUD (Create, Read, Update, Delete)** application for managing products, built with:

- **React** (Frontend UI)
- **Tailwind CSS** (Styling)
- **Axios** (API requests)
- **React Router DOM** (Routing)
- Backend API (Node.js + Express + MongoDB)
